This article offers a sample of basic diagram usage that can be used in Hugo content files.

Please see also Mermaid.

Usage

Via Shortcode

1YOUR DIAGRAM INSTRUCTIONS

You can also wrap the shortcode with other shortcodes, such as text/align-center.

1{{% text/align-center %}}
2YOUR DIAGRAM INSTRUCTIONS
3{{% /text/align-center %}}

Mermaid code block

1```mermaid
2YOUR DIAGRAM INSTRUCTIONS
3```

Examples

Flow Chart

1graph TD
2    A[Christmas] -->|Get money| B(Go shopping)
3    B --> C{Let me think}
4    C -->|One| D[Laptop]
5    C -->|Two| E[iPhone]
6    C -->|Three| F[fas:fa-car Car]

graph TD A[Christmas] –>|Get money| B(Go shopping) B –> C{Let me think} C –>|One| D[Laptop] C –>|Two| E[iPhone] C –>|Three| F[fas:fa-car Car]

Sequence Diagram

1sequenceDiagram
2    Alice->>+John: Hello John, how are you?
3    Alice->>+John: John, can you hear me?
4    John-->>-Alice: Hi Alice, I can hear you!
5    John-->>-Alice: I feel great!

sequenceDiagram Alice-»+John: Hello John, how are you? Alice-»+John: John, can you hear me? John–»-Alice: Hi Alice, I can hear you! John–»-Alice: I feel great!
